Git Bash : aws서버에 접속을 하여 명령을 실행하는 프로그램
1. python 파일(서버)를 실행하기 위해 아래와 같은 명령어 입력하여 접속한다.
 : ssh -i 'key_pem(끌어다 놓기)'@'내ip주소'
2. 가상환경을 킨다.
3. 몽고DB설치(해당서버에서 몽고DB를 이용한다면 설치해야함)

4. 그 외 관련 모듈필요한게 있다면 설치한다
pip install 모듈이름 (ex)beatifulsoup4, request 등등..)

5. 접속시 5000포트 -> 80포트로 바꿔주는 명령어
 1) GitBash에서 해당 명령어 실행
 2) 인바운드 규칙에서 80포트를 추가

6. GitBash를 꺼도 서버가 24시간 돌아갈수 있도록 하는 환경설정해준다.
